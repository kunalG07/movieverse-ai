# MovieVerse AI

MovieVerse AI is a machine learning based web application that predicts whether a movie will be a Hit, Average, or Flop.

## Features
- Movie success prediction using ML
- Confidence score output
- FastAPI backend
- Modern frontend using HTML, CSS, JavaScript

## Project Structure
- Frontend: website files
- Model: ML model, encoder, dataset, and API

## Tech Stack
- Python
- FastAPI
- Scikit-learn
- HTML
- CSS
- JavaScript
